My Vim Enviroment
========

build up your vim enviroment using neobundle.
In addition, my favorite configure of vim (.vimrc) will be put your home directory

Quick Start!
=======
    make setup

Update
======
    make update

Use
====
    vim .vimrc
    :neoBundleInstall
